Fitur Unggulan

Semua yang dibutuhkan
untuk ujian sekolah

Dirancang khusus untuk kondisi sekolah Indonesia — tidak bergantung internet, ringan, dan mudah dioperasikan guru tanpa keahlian IT.

📡
100% Offline & Lokal
Server berjalan di komputer guru sendiri. Tidak perlu internet saat ujian — data tidak keluar dari sekolah.
📱
Siswa Pakai HP / Laptop
Siswa cukup buka browser dan ketik alamat IP yang ditulis di papan tulis. Tidak perlu install aplikasi apa pun.
⚡
Langsung Jalan
Dobel klik Jalankan-CBT.bat — server langsung aktif dalam hitungan detik. Tidak perlu konfigurasi.
🧮
Soal dengan Rumus & Gambar
Editor soal mendukung rumus matematika (KaTeX), gambar (paste/drag-drop/upload), pilihan ganda, dan soal uraian.
🖥️
Monitor Ujian Real-Time
Pantau siapa yang sudah mengerjakan, berapa soal yang dijawab, dan paksa selesai ujian langsung dari dashboard guru.
📊
Nilai Otomatis & Analisa
Penilaian pilihan ganda otomatis, rekap nilai per kelas, analisa tingkat kesulitan per butir soal, dan cetak laporan.
🖨️
Cetak Dokumen Resmi
Rekap nilai, daftar hadir, dan berita acara ujian bisa dicetak langsung dengan kop surat sekolah otomatis.
💾
Backup Sangat Mudah
Seluruh data — soal, nilai, siswa, logo, KOP surat — tersimpan dalam satu file eschool.db. Backup cukup copy satu file.
🔐
Sistem Lisensi per Sekolah
Lisensi terkunci ke NPSN sekolah — tidak bisa dipindahtangankan. Nama sekolah tampil otomatis di seluruh dokumen.
